ゲームタイトル: Haruchan Cannonball

概要:
このゲームはPythonとpygameを使用して制作された2Dゲームです。
プレイヤーはハル（というよりは、ハルを射出するラビ）を操作して、ビル屋上に置かれたバナナを獲得し、より高いスコアを目指します。

操作方法:
- ENTERキーでゲームの開始/仰角の決定/パワーの決定を行います。
- 上下キーで仰角の調整を行います。
- hキーでhelpを表示します。

動作環境:
- Python 3.13
- pygame 2.6.1 以上

インストール方法:
1. Python をインストールしてください。
2. pygameをインストールしてください:
   pip install pygame
3. ゲームを起動してください:
   python game.py
